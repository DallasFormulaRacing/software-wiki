# Introduction

Unfortunately, as goes for many schools, UTD does not (in most cases) provide the needed learning material for many of the technologies that are utilized within [DFR](https://dallasformularacing.com/) ([Dallas Formula Racing](https://dallasformularacing.com/)). The same goes for many of the technologies that you will likely end up using in your professional life. Thus the goal of this wiki, and primarily its onboarding, is to give you everything you need to know to get up and running within DFR Software, and likewise to give you a solid understanding of the projects that we work on and the technologies that we use.
