# Your first project

Now that you've got a general understanding of how to use [Git](https://git-scm.com/) and [Python](https://www.python.org/), it's time we get started with your first project! This a super simple data visualization project that will test some basic skills using [Python](https://www.python.org/) packages and your general coding knowledge.

## Getting started

- [ ] 1. Fork [this repository](https://github.com/DallasFormulaRacing/Onboarding) to your own GitHub account.
- [ ] 2. Clone the repo that you just forked to your local machine.
- [ ] 3. Install the two required packages for the project, which are `pandas` and `plotly`.
- [ ] 4. Download <a href="../../assets/can_data.csv" download="can_data.csv">this CSV</a> to your local machine and place it in the repo folder that you cloned from step one.

## The project

There are 5 main tasks that you will need to complete for this project, which are as follows:

- [ ] 1. Load the CSV file into a `pandas` DataFrame.
- [ ] 2. Clean the data by removing all columns named 'Analog Input' that contain the same value for every row.
- [ ] 3. Create a line graph(s) using `plotly` that shows the change in `RPM` over time and change in `TPS` over time. This can either be done in separate graphs or in the same graph if you prefer.
- [ ] 4. Add appropriate titles and labels to your graphs for clarity.
- [ ] 5. Submit a pull request with your completed code to the original repository.

Once you have completed the above, notify Arjun (`@sq.id`) or Will (`@Skribb11es`) on Discord of your completion and explain why you want to join the team. We will go over your work, and provide feedback where needed. If all looks good, then welcome to the team!
